在按照Exam Grading 自动测试评分系统使用教程搭建 rustlings 做题环境时遇到了一些问题，这里记录一下解决方法。

准备 git 环境和获取 exam-grading 代码的步骤就不再进行赘述，从添加子模块开始。

添加子模块

1、首先将上一步创建的exam-grading仓库文件拉到本地，比如我放在 ~/2024winter 目录下。

2、安装 rustlings（自己按照安装教程的指导安装），安装完后找一个目录（我同样放在 ~/2024winter 目录下）执行 rustlings init 。

3、在 github 再创建一个空的私有仓库，之后将 rustlings 内容推送到该仓库（空仓库会有指导该怎样推送）。

5、再将 exam-grading 新增的修改推送到远程仓库。
